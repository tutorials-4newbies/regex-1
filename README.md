# Setup

Clone this repo to your pc   
```bash
 git clone https://github.com/tutorials-4newbies/regex-lessons.git
 ```

create a virtualenv   
**Linux:** 
```bash
python -m venv venv
source testing_venv/bin/activate
```

**Windows**
```
python -m venv venv
testing_venv\Scripts\activate
```

After that
```bash
pip install -r requirements.txt
```
